# Frontend (Vite + React)

This is a minimal Vite + React template created for the project.

Quick start:

1. Install dependencies (npm or pnpm/yarn):

```bash
cd frontend
npm install
# or with pnpm
# pnpm install
```

2. Start dev server:

```bash
npm run dev
```

The app will be available at the address shown by Vite (usually http://localhost:5173).
